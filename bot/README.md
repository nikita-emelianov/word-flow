# Word Flow bot

## Fast start
Use [Word Flow manual](https://github.com/nikita-emelianov/word-flow#readme) for a fast start

## Local development
1. Install packages
```bash
npm install
```
2. Setup .env variables in '.env' (see [.env.defaults](.env.defaults))
3. Build and run with
```bash
# build
npm run build

# start
npm run start
```
4. Use bot
