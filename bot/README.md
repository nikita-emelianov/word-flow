# Word Flow bot

## HOWTO start
Use [Word Flow manual](../README.md) for a fast start

## Local development
1. Install packages
```bash
npm install
```
2. Set up .env variables in '.env' (see [.env.defaults](.env.defaults))
3. Run with
```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```
4. Use bot
