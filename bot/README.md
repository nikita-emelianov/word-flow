# Word Flow bot

## Fast start
Use [Word Flow manual](https://github.com/nikita-emelianov/word-flow#readme) for a fast start

## Local development
1. Install packages
```bash
$ npm install
```
2. Set bot `token` to `new Telegraf(...)` in `main.ts`.
3. Run with
```bash
$ npm run start
```
4. Use bot
